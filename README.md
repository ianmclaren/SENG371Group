# DimSpace

## SENG371 Spring 2023 - Group 4

## Run Locally

### Prerequisites

- [Node.js](https://nodejs.org/en/)

### Installation

Enter the correct folder `dimspace/` and install the dependencies:

```bash
cd dimspace
npm install
```

### Run

```bash
npm start
```
